# Screenshots

Place actual project screenshots here and update the table in the root `README.md`.

## Naming Convention

| Filename | Content |
|----------|---------|
| `banner_placeholder.png` | Project banner / header image |
| `01_hardware_assembly.jpg` | Full breadboard/PCB assembly photo |
| `02_door_opening.jpg` | Physical door in motion |
| `03_lcd_display.jpg` | LCD showing "Welcome" or "Access Denied" |
| `04_rfid_grant.jpg` | RFID card being scanned, green LED on |
| `05_dashboard.jpg` | Web dashboard showing live status |
| `06_serial_log.jpg` | Arduino Serial Monitor output |
| `07_event_log.jpg` | /api/logs JSON or web table |
| `08_schematic.jpg` | Fritzing or hand-drawn circuit diagram |

## Tips

- Use 1280×720 or higher resolution
- Consistent lighting (natural or softbox)
- Include a label/caption in each photo
- Crop to remove distracting backgrounds
- GIF of the door opening sequence makes a great `demo.gif` for the README
